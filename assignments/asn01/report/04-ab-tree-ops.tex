\subsection{Deletion}

From the tree above, 309 is deleted with the following steps.

First, we kick out 309 from the root node.
\begin{center}
	\begin{tikzpicture}[every node/.style={draw, rounded rectangle},
						level distance=8mm,
						level 1/.style={sibling distance=40mm},
						level 2/.style={sibling distance=20mm}
					   ]
		   \node[rounded rectangle, red] {$(!!):703$}
			child { node[rounded rectangle] {$210$} 
				child { node[rounded rectangle] {$55$} }
				child { node[rounded rectangle] {$262:269$} }
			}
			child { node[rounded rectangle] {$548$} 
				child { node[rounded rectangle] {$449:513$} }
				child { node[rounded rectangle] {$608:656$} }
			}
			child { node[rounded rectangle] {$745$} 
				child { node[rounded rectangle] {$733$} }
				child { node[rounded rectangle] {$795:846$} }
			};
	\end{tikzpicture}
\end{center}

Then, we pick $548$ to be the successor, steal it out of its old node, then bring it up to replace the empty slot.
\begin{center}
	\begin{tikzpicture}[every node/.style={draw, rounded rectangle},
						level distance=8mm,
						level 1/.style={sibling distance=40mm},
						level 2/.style={sibling distance=20mm}

		   \node[rounded rectangle] {$548:703$}
			child { node[rounded rectangle] {$210$} 
				child { node[rounded rectangle] {$55$} }
				child { node[rounded rectangle] {$262:269$} }
			}
			child { node[rounded rectangle, red] {$(!!)$} 
				child { node[rounded rectangle] {$449:513$} }
				child { node[rounded rectangle] {$608:656$} }
			}
			child { node[rounded rectangle] {$745$} 
				child { node[rounded rectangle] {$733$} }
				child { node[rounded rectangle] {$795:846$} }
			};
	\end{tikzpicture}
\end{center}

Finally, we $608$ to be the successor, steal it out of its old node, then bring it up to replace the empty slot.
\begin{center}
	\begin{tikzpicture}[every node/.style={draw, rounded rectangle},
						level distance=8mm,
						level 1/.style={sibling distance=40mm},
						level 2/.style={sibling distance=20mm}
					   ]

			child { node[rounded rectangle] {$210$} 
				child { node[rounded rectangle] {$55$} }

			}
			child { node[rounded rectangle] {$608$}

			}
			child { node[rounded rectangle] {$745$}

				child { node[rounded rectangle] {$795:846$} }
			};
	\end{tikzpicture}
